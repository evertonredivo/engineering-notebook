# Engineering Notebook

> An evidence-based engineering handbook for studying computer systems from first principles.

---

## Overview

Engineering Notebook is a long-term technical documentation project dedicated to the systematic study of computer systems through observation, experimentation, analysis, and evidence.

The repository documents reproducible engineering investigations covering computer networks, operating systems, Linux, computer architecture, information security, cloud infrastructure, distributed systems, digital forensics, and related disciplines.

The objective is to develop technical understanding rather than collect isolated information.

---

## Engineering Philosophy

The project follows four fundamental principles.

- Evidence before conclusions.
- Observation before modification.
- Reproducibility before publication.
- Documentation as part of every experiment.

Additional engineering principles are defined in the project manifesto.

## Repository Structure

This repository is organized into independent engineering laboratories.

Each laboratory contains:

- research plan
- theoretical foundation
- experiments
- evidence
- technical analysis
- conclusions
- references

---

## Primary Sources

Whenever possible, technical conclusions shall be supported by primary sources, including:

- RFCs
- IEEE Standards
- ISO Standards
- Linux Kernel Documentation
- Vendor Documentation
- Source Code
- Peer-reviewed Publications
